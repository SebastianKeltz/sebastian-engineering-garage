Sebastian Keltz Engineering Garage Website

UPLOAD THESE TO GITHUB:
- index.html
- styles.css
- script.js
- README.txt
- assets folder

Do NOT upload only the ZIP file. Unzip it first, then upload the files above.

The most important files are:
1. index.html = homepage
2. styles.css = website design
3. script.js = mobile menu
4. assets/Sebastian_Keltz_Resume.pdf = resume link

How to publish with GitHub Pages:
1. Upload the files to your repository.
2. Click Commit changes.
3. Go to Settings.
4. Click Pages.
5. Under Source, choose Deploy from a branch.
6. Choose main and /root.
7. Click Save.
8. Wait 1-3 minutes for the website link.
