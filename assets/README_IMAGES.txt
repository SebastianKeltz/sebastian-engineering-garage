Sebastian Keltz Portfolio Image Guide
=====================================

Place all images and the resume PDF in this assets folder.

Important:
- Keep the filenames exactly as listed below if you want the placeholders to update automatically.
- Use JPG for photos and PNG for screenshots if needed.
- Compress large images before uploading so the site stays fast on GitHub Pages.
- Recommended image width: 1400 to 2200 pixels for large photos, 900 to 1400 pixels for smaller cards.
- Recommended file size: under 500 KB per image when possible.

Resume
------
Sebastian_Keltz_Resume.pdf
The resume buttons in index.html already link to this file.

assets/reports/

Photo Tips
----------
- Crop photos so the subject is obvious.
- Use bright, clear images when possible.
- Recruiters should be able to quickly understand what the photo proves.
- Avoid uploading unsafe wiring photos without context. If a photo shows an in-progress setup, explain what was being tested.
- Do not upload exams, quizzes, answer keys, professor slides, Canvas files, homework solutions, or lab manuals unless you have explicit permission.
