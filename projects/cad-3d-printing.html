Project Case Study

CAD & 3D Printing for Project Prototyping

Using Fusion 360 and 3D printing to turn project problems into physical parts that can be tested, measured, and revised.

View Resume Design Process

Overview

I use CAD and 3D printing as part of my project workflow. Instead of only designing parts on screen, I use printed prototypes to check fitment, mounting points, clearances, and serviceability on real builds.

Engineering Goal

  • Create practical parts for real project constraints.
  • Use CAD to plan clearances, mounting points, and service access.
  • Use 3D printing for fast prototype testing and revision.
  • Connect mechanical design choices to electrical project needs.
  • Document how parts change after fit checks and testing.

Materials / Parts Used

  • 3D printed prototype parts.
  • Battery support and packaging concepts.
  • Mounting brackets and fitment parts.
  • Water bottle mount holder.
  • Project hardware used for fit checks.

Tools Used

Fusion 360 3D Printer Calipers Hand Tools Test Fitting Project Measurement

Build Process / Timeline

From measurement to revision.

01

Measure

Take real measurements from the project instead of guessing from the screen.

02

Model

Build the part in CAD around the actual constraints, mounting points, and clearances.

03

Print

Make a physical prototype that can be held, installed, and inspected.

04

Test fit

Check clearances, strength, mounting, and serviceability on the real project.

05

Revise

Improve the design based on what did not fit or work well.

Testing and Troubleshooting

  • Checked printed parts against the actual project hardware.
  • Looked for clearance issues, weak areas, and hard-to-service geometry.
  • Compared CAD assumptions against physical fitment.
  • Revised dimensions when a part did not fit or support the build well enough.
  • Documented what should change before the next print.

What I Learned

I learned that a CAD model is only the first version of the idea. The real test happens when the printed part has to fit around wires, hardware, batteries, frames, and real-world constraints.
